= Contributing

We welcome any contributions to the Swedish Reference Genome Portal. 

The most common contribution is to submit a species to the portal. If you are interested in doing that, 
you can contact us at dsn-eb@scilifelab.se and we will guide you through the process.

It is also possible to submit a species through a Pull Request. For a detailed guide on how to do that, please see:

* link:species_submission/how_to_submit_through_pull_request.md[species_submission/how_to_submit_through_pull_request.md]
